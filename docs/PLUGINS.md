## Plugins NOT to Install

These plugins are unnecessary for a content site and add bloat:

- WooCommerce (unless selling digital products — use Gumroad instead)
- Jetpack (too heavy, individual features available separately)
- Yoast SEO (replaced by Rank Math)
- W3 Total Cache (replaced by WP Rocket)
- Multiple page builders (Elementor only)
- Contact Form 7 (replaced by WPForms)
